# scikit-learn

## Source Type
Documentation / Repository

## Main Use
Primary practical source for machine-learning concepts, algorithms, estimators, preprocessing, model evaluation, and examples.

## Best For
- supervised learning
- unsupervised learning
- model evaluation
- preprocessing
- pipelines
- estimator API
- practical examples

## Not Ideal For
- deep mathematical derivations
- from-scratch implementation details
- neural network internals
- very theoretical statistics

## Usage Rule
Use scikit-learn as the practical anchor for ML entries. For purely mathematical topics, combine it with a more suitable statistics or mathematics source.
